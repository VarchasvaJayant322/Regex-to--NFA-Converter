# Regex-to--NFA-Converter
A simple tool to convert a regular expression to ε-NFA. Additionally , you can also convert the NFA to minimized DFA with each step involved described in detail.

The tool consists of the following featues:
1. Converts Regular Expressions → ε-NFA using Thompson’s Construction.
2. Interactive visualization of automaton (states, transitions, ε-moves).
3. String acceptance tester with step-by-step simulation.
4. ε-closure inspection for understanding non-determinism.
5. Converts ε-NFA → DFA and performs DFA minimization with each step described in detail.
6. Step-by-step construction of automaton from regex.
7. Parse tree construction for Structural decomposition of your regular expression.
